2) Via mensajes usando al servidor como mediador para comunicacion entre los clientes (detalles mas abajo) -> SOLUCION MAS PROBABLE.

DETALLES:
2)
- Los Clientes la opciones de conexiones que el Servidor también tiene (Global y otros usuarios, excluyendo al propio Usuario)
- Cada que se conecta o desconecta un Cliente, el Servidor envia un mensaje avisando que un cliente se ha conectado o desconectado.
- Con cada mensaje de conexion o desconeccion las opciones se actualizan.
- El Cliente selecciona una opcion (Global u otro Usuario) de la barra de opciones y el Servidor lo redirige segun corresponda.
	* Si fue Global, entonces dira "Mensaje de {USER}: {MESSAGE}". TODOS lo podran ver.
	* Si fue a uno en especifico dira: "De {USER_SEND} A {USER_REC}: {MESSAGE}". SOLO los involucrados lo podran ver (Server incluido).
	* El servidor también puede ser seleccionado como un Usuario para enviar mensajes.

COMO HACERLO:
- Cuando un cliente se conecta con el servidor, el servidor envía un mensaje de todos los clientes conectados actualmente. Los clientes reciben ese mensaje y actualizan la lista de conectados, actualizando así los botones con sus respectivas conexiones.
- Cuando un cliente se desconecta del servidor, el servidor envía un mensaje de todos los clientes conectados actualmente. Los clientes reciben ese mensaje y actualizan la lista de conectados, actualizando así los botones con sus respectivas conexiones.
- Los clientes tienen un boton, el cual conecta con el servidor. Funciona igual que el resto de botones, pero ese es inamovible, lo cual significa que al presionar el boton "Eliminar Desconectados", este no eliminra el boton del servidor, si está desconectado.
- Los clientes pueden seleccionar uno o mas botones de clientes. Al presionarlos enviaran un mensaje al servidor. El servidor recibe los menssajes y los envia a los respectivos usuarios.
- Si el cliente elige el mismo cliente, este se elimina de la lista de destinatarios.
- Si la lista de destinatarios está vacía, entonces es un mensaje "global".
- Si se elige la opcion "global", entonces se elimina el resto de destinatarios, quedando solamente "global".
- Si se elige un destinatario, estando "global" dentro de la lista de destinatarios, se elimina "global" en la lista de destinatarios.
