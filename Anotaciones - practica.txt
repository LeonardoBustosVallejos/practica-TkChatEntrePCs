2) Via mensajes usando al servidor como mediador para comunicacion entre los clientes (detalles mas abajo) -> SOLUCION MAS PROBABLE.

DETALLES:
2)
- Los Clientes la opciones de conexiones que el Servidor también tiene (Global y otros usuarios, excluyendo al propio Usuario)
- Cada que se conecta o desconecta un Cliente, el Servidor envia un mensaje avisando que un cliente se ha conectado o desconectado.
- Con cada mensaje de conexion o desconeccion las opciones se actualizan.
- El Cliente selecciona una opcion (Global u otro Usuario) de la barra de opciones y el Servidor lo redirige segun corresponda.
	* Si fue Global, entonces dira "Mensaje de {USER}: {MESSAGE}". TODOS lo podran ver.
	* Si fue a uno en especifico dira: "De {USER_SEND} A {USER_REC}: {MESSAGE}". SOLO los involucrados lo podran ver (Server incluido).
	* El servidor también puede ser seleccionado como un Usuario para enviar mensajes.


-> IMPORTANTE:
AMBOS:
	* SE AMPLIARA EL ANCHO DE LA PANTALLA DEL PROGRAMA, DIVIDIENDOLO EN 2 PARTES.
	* LA PARTE DERECHA SE MANTENDRA COMO ESTA ACTUALMENTE.
	* LA PARTE IZQUIERDA SERÁ EXCLUSIVAMENTE PARA NUEVAS CONEXIONES.
	* ACTUALIZAR LA ELECCION DE USUARIOS PARA ENVIAR MENSAJES, EN VEZ DE UN MENU DE OPCIONES SERÁN BOTONES QUE APARECERAN DENTRO DE UN ESPACIO DELIMITADO.
	* ESTOS BOTONES TENDRÁN EL NOMBRE DEL USUARIO CONECTADO AL CUAL REPRESENTAN.
	* A LA DERECHA DEL NOMBRE, HABRÁ UN CUADRADO, DIBUJADO POR EL PROPIO PROGRAMA, DE COLOR ROJO O VERDE.
	* EL COLOR VERDE CORRESPONDE SI EL USUARIO ESTA CONECTADO.
	* EL COLOR ROJO CORRESPONDE SI EL USUARIO ESTA DESCONECTADO.
	* AHORA, LA OPCION "GLOBAL" SERÁ UNA OPCION QUE APARECE EN EL CUADRO DE LA IZQUIERDA.
	* EN VEZ DE UN MENU, LOS NUEVOS USUARIOS APARECERAN BOTONES QUE APARECEN EN LA PARTE IZQUIERDA DE LA PANTALLA.
	* HACIENDO CLICK SE CAMBIARA EL COLOR DEL BOTON A UN TONO MAS OSCURO, SEÑALANDO QUE ESE CLIENTE FUE FUE ELEGIDO PARA ENVIARLE UN MENSAJE.
	* HACER CLICK UNA SEGUNDA VEZ SOBRE ESE USUARIO HARA QUE VUELVA A LA NORMALIDAD Y NO SE LE ENVIA UN MENSAJE.
	* CUANDO UN CLIENTE SE DESCONECTE Y SE INTENTE ENVIAR UN MENSAJE, LA OPCION QUEDA ELIMINADA.
	* AGREGAR UN BOTÓN DE "ELIMINAR DESCONECTADOS". AL HACER CLICK EN ESE BOTON SE ELIMINAN COMO OPCIONES A LOS USUARIOS DESCONECTADOS.

CLIENTE:
	* AGREGAR UN BOTON DE "RECONECTAR", QUE NORMALMENTE ESTARÁ DESHABILITADO
	* SOLO SE ACTIVARÁ EL BOTÓN CUANDO EL SERVIDOR SE CALLO (SE DESCONECTO)
	* AL PRESIONARLO VOLVERÁ A INTENTAR LA CONEXIÓN, SI 5 SEGUNDOS DESPUÉS NO HUBO RESPUESTA LANZA UN MENSAJE DE ERROR.
